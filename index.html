About this project

Welcome to the start page of Wonder Fun! This project is built on 2020/5/9，It is an open source project focusing on the form of community.

Its use is very simple, just need to The warehouse of this project 上提交 PR，即可创造一个属于自己的导航页面。访问它只需要一个 URL 后缀即可直达，无需注册和登录，非常方便易用。

现有的类似站点已经很多了，为什么我仍然选择编写了这个项目？这是因为它们都是闭源的私人项目，只有作者本人才可以进行维护。

与此同时，这些站点的导航项大多已经预设。想要修改它们，要么需要注册后修改，要么就干脆不给改，个人觉得略显鸡肋。如果你更换了电脑，不就需要重新登录了吗？

介于采用 GitHub 托管的形式，本站属于“无后端”项目，一些可以提升用户体验的功能需要借助后端才能完成，我暂时没有去实现它。
