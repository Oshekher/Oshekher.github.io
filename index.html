About this project

Welcome to the start page of Wonder Fun! This project was built on 2020/5/9 and is an open source project focusing on the community.

its use is very simple requiring only the item repository submission PR, you can create your own navigation page. It only needs a URL suffix to access it directly without registration and login, which is very convenient and easy to use.

There are already many similar sites, why did I still choose to write this project? This is because they are all closed-source private projects that only the author can maintain.

At the same time, most of the navigation items for these sites have been preset. If you want to modify them, you need to modify them after registration, or you just don't want to modify them. I think it's a bit tasteless. If you change your computer, don’t you need to log in again?

Because it is hosted by GitHub, this site is a "no backend" project. Some functions that can improve user experience need to be completed with the help of the backend. I have not implemented it for the time being.
